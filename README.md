# utha-ui

Check storybook [here](https://ytanruengsri.github.io/utha-ui)

## Setup
```
$ yarn
```

## How to start?
```
$ yarn run storybook
```

open `http://localhost:9001` in your favourite browser

## How to deploy to gh-page?
- Push latest changes to master and keep master clean
- run `yarn run deploy`
